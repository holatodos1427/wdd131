REQUIRED IMAGE FILES
====================

Place the following WebP images in this folder:

1. kyoto-small.webp
   - Used for mobile screens (default src)
   - Recommended dimensions: 800 x 500 px
   - Subject: Fushimi Inari torii gates or Kyoto temple scene

2. kyoto-large.webp
   - Used for screens 800px+ wide (via <source media="(min-width: 800px)">)
   - Recommended dimensions: 1400 x 700 px
   - Same or related subject, wider crop/higher resolution

WHERE TO GET IMAGES:
  - https://www.pexels.com (search "Kyoto" or "Fushimi Inari")
  - https://unsplash.com (search "Kyoto Japan")

CONVERSION TO WebP:
  - Use squoosh.app or any image editor to convert JPG/PNG → WebP

LICENSE NOTE:
  Pexels and Unsplash images are generally free to use without
  attribution, but always verify the individual photo's license.
